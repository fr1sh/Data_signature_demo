数据签名靶场

加签程度与安全测试难度之间存在紧密关系，不同的加签程度会对安全测试提出不同的挑战。以下是对比不同加签程度与安全测试难度的考虑。
|编号| 加签程度 | 测试难度 |
|----------|----------|----------|
|   1  |   无签名           |     ⭐️     |
|   2  |   有签名未混淆       |    ⭐️⭐️      |
|   3  |   有签名半混淆       |    ⭐️⭐️⭐️      |
|   4  |   有签名全混淆       |    ⭐️⭐️⭐️⭐️     |

以登录页面为例，在登录口是否进行加签，影响登录爆破的结果为例
